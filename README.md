# ET Hardware Runtime

This repository is a WIP and is intended to allow Eye tracking hardware to work on other operating systems which are not supported by official drivers.

## Feats before being seen in usable state
- Successfull communication with ET hardware (Tobii Eye Tracker 5 as starting point)
- IPC server with ET data (Gaze point and Gaze origin)
- Service creation for use as Daemon
- Opentrack tracker


This repo is not expected to be usable for a while, and is considerd purely in development
